we‑term — how to use it

Keyboard

Tap the terminal to show or hide the on‑screen keyboard. Tap it again to dismiss.

  • Sym switches to symbols; ABC switches back.
  • Backspace is at the top‑right; Shift is on the left.
  • Space and Enter are on the bottom row. Esc and Tab are in the button bar.

Button bar

The row above the keyboard holds extra keys:

  • ⚙ opens Settings.
  • Ctrl / Meta — tap to arm, then tap a key (e.g. Ctrl then C sends Ctrl‑C).
  • Sel selects text, Paste pastes, Tab, PgUp/PgDn, and the arrow keys.
  • Scrl — in GNU screen, enter copy/scrollback mode (Ctrl‑A [), then use PgUp/PgDn and arrows to scroll back; Esc exits. See docs/SCROLLBACK.md.

Autocomplete

As you type on the on‑screen keyboard, a suggestion bar appears above the buttons with matching commands from your history and common commands. Tap one to fill it in — a trailing space is added automatically. (Available with the on‑screen keyboard.)

Copy & paste

Long‑press the terminal (or tap Sel) to select text in place. Drag the handles to adjust, then tap Copy on the menu iOS shows. Tap elsewhere to dismiss. To paste, tap Paste, long‑press the box and choose Paste, then tap Send. Images can be pasted too.

Settings (⚙)

  • Blink cursor — toggle the cursor blink.
  • Haptic feedback — vibrate on key taps.
  • Use system keyboard — use your device’s built‑in keyboard instead of the on‑screen one.
  • Command completion — show the autocomplete suggestion bar.
  • Glide typing — drag across the letters to glide‑type a word; tap a suggestion to insert it.
  • Button bar — choose which keys appear in the bar above the keyboard.

System keyboard

When Use system keyboard is on, tap the terminal to bring up your device keyboard. A ⚙ appears in the top‑right — tap it to open Settings and switch back to the on‑screen keyboard.
